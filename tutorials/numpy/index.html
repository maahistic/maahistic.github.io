Vectorized Operations,
Made Visual

NumPy replaces slow Python loops with contiguous memory blocks and CPU-native SIMD instructions, letting you operate on millions of numbers in a single call. Every section below is a live playground — drag sliders, click buttons, and watch arrays transform in real time.

The Core Idea

A standard Python loop touches one value at a time: read → type-check → compute → store → repeat. NumPy skips the overhead entirely. It allocates a flat, typed memory buffer and hands the math to compiled C/Fortran routines that process entire vectors in a single CPU instruction. The result? 10–100× speed-ups with cleaner code.

Mini Check

3 quick questions to confirm understanding

Q1. What is the .shape of np.zeros((2, 4, 3))?

np.zeros creates an array of the exact shape you pass. A tuple (2, 4, 3) defines 2 layers × 4 rows × 3 columns — the shape is preserved as-is.

Q2. Broadcasting (3,1) + (1,4) produces shape:

Broadcasting aligns from the right. Axis 1: 1 stretches to 4. Axis 0: 3 stays, 1 stretches to 3. Result: (3, 4).

Q3. np.arange(12).reshape(3,4).sum(axis=0) returns shape:

axis=0 collapses rows (3 rows vanish), leaving one value per column. 4 columns → shape (4,).
